Audacity Glitch Art

This is a method I use quite a bit on here, there's plently of fun stuff to do with it! At its core, Audcity has the ability to completely rewrite the raw file data of images. This can be done with all the typical tools within the platform such as reverb, echo, various filters, pitch shifting etc.

    Effects

  • Echo: The most popular effect. This will create mirages and duplicate images depending on what settings you have.
  • Reverb: Similar to Echo but It generally ads some rainbow color distortion as well. Feels a bit more dreamy.
  • Pitch Shifter: The best I can tell this generally handles color. It can break your image if you go too hard with it so take it slow.
  • HP / LP Filter: Also focuses on color. Allows colors of a certain value to pass through the edit.
  • Insert > Noise: I really like throwing bits of noise into images for a super fuzzy glitchy look. Bonus points if you hit the noise with effects afterwards.

    Shortcuts: These really help speed up the process if you're trying to make a full glitch photo set or a gif out of a ton of images. Took me way to long to start using them.

  • Repeat last effect: CTRL R
  • Export: CTRL SHIFT E
